Características

Agrega nuevos contactos

Guarda información básica como fecha de nacimiento, localidad, estado y pais

Todos los datos son almacenados en una base de datos local SQLite, bajo la capa de abstracción de Android Room

Añade anotaciones

La base de datos local está diseñada para admitir infinitas anotaciones personalizadas de cada contacto.

Estas anotaciones constan de título, descripción y fecha de creación

Filtra los contactos

El menú principal cuenta con un potente buscador en tiempo real que te permite filtrar de forma rápida y visual los contactos por sus apodos

Añade sucesos...

Para anotar cualquier cosa que ciertas personas indeseables te hayan hecho

...y asócialos a las personas culpables

Para dejar constancia de quién fue el cabezilla y quienes ayudaron

Crea grupos y rellénalos de contactos

Para agrupar rápidamente un conjunto de contactos y sucesos bajo un mismo paragüas

¿Quién se llevaba con quién? ¡Apúntalo y no te olvides!

Con pulsar unos pocos botones podrás recordar las relaciones entre tus contactos, porque aquí nadie quiere muy cerca a gente que se relacione con gentuza, ¿verdad?

Rendimiento excelente con multihilo

Gracias a las corrutinas de Kotlin, Spylook puede entregar una velocidad sin igual en la carga de datos


Todas las consultas a la base de datos local se realizan en segundo plano, dejando el hilo principal encargado de la UI libre de posibles bloqueos e interrupciones de la experiencia de usuario
